Neutron

A collection of Sass mixins to ease our styling development.

View on Github

Examples

Easy Style Reset

                    
@include Reset {
    body {
        @include SansSerif;
        line-height: 1.65;
        color: #111;
    }
    a {
        color: #38c;
        &:hover {
            color: lighten(#38c, 10%);
        }
    }
    img {
        max-width: 100%;
    }
}

Article Rendering (with normalize.css) 


@include ResetTitle {
    @include Smoothing;
};
body {
    @include SansSerif;
    line-height: 1.65;
}

Responsive Card


.card {
    width: 50%;
    @include Ratio(1.2);
}

Ellipsis


.oneline {
    @include Ellipsis;
}
.multiline {
    @include Clamp(3);
}
